# The Hearth

A warm, local-first family attentiveness web app.

## Run locally
1. `npm install`
2. `npm run dev`

## Build
- `npm run build`
- `npm run preview`

## Deploy to Vercel
- Push this repo to GitHub.
- Import the repo in Vercel.
- Framework preset: **Vite**.
- Build command: `npm run build`
- Output directory: `dist`

## MVP features
- Today, People, Asks, Curriculum screens.
- Seed data for Aarav, Aanya, Manju.
- Quick capture and CRUD actions.
- Conversation prep guidance by context.
- localStorage persistence.
- JSON export/import.
- Basic web app manifest for installability.

## Notes
All data is local in your browser for v1. No backend, no auth.
